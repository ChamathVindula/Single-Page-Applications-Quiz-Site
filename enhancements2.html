Enhancement 1
Quiz Timer Triggered on Id Input

  • I have implemented a timer for the quiz where a user has to complete the quiz within the allocated time.
  • The timer function is triggered when the user enters his/her id and moves over to anwser the immediate question.
  • This occurs when the users enters the id and as soon as the id input looses focus.
  • To create this timer I had to create a setInterval timing event where a certain function can be executed repeatedly after a specified no. of milliseconds.
  • The nested function repeatedly takes the difference of two time values and displays the result.
  • As soon as the no. of minutes and no. of seconds turns to '0' then the quiz is terminated
  • Once quiz is terminated the entire process is bypassed and it moves over to the result.html page (note that no. of attempts will not be increased)
  • In the case of a 'time over' the user can retry the quiz without loosing his/her number of attempts
  • I have used the web site 'w3schools' to gain the basic understanding and key features required to implement the timer. The source is cited below:
    w3schools (2016),JavaScript Timing Events, viewed 23 April 2018, Available at: https://www.w3schools.com/js/js_timing.asp
